# 🔍 상품 추천 시스템 (TF-IDF 기반)

## 📌 프로젝트 개요

이 프로젝트는 **TF-IDF와 코사인 유사도**를 활용하여
사용자가 입력한 검색어와 유사한 상품을 추천하는 간단한 검색 시스템입니다.

---

## ⚙️ 핵심 개념 (쉽게 이해하기)

### 1️⃣ TF-IDF란?

👉 문장을 **숫자 벡터로 변환**하는 방법입니다.

예시:

```
"아이스 아메리카노" → [0.1, 0.8, 0.3 ...]
"콜드브루 커피" → [0.2, 0.7, 0.4 ...]
```

✔️ 비슷한 문장일수록 → 벡터도 비슷해집니다.

---

### 2️⃣ 코사인 유사도 (Cosine Similarity)

👉 두 벡터가 얼마나 비슷한지 계산합니다.

* **1에 가까울수록** → 매우 유사
* **0에 가까울수록** → 관련 없음

---

### 3️⃣ 추천 시스템 동작 흐름

```
"아아" 입력
   ↓
벡터로 변환
   ↓
전체 상품과 유사도 비교
   ↓
유사도 높은 TOP 5 선택
```

---

## ⚠️ 중요한 한계 (핵심 포인트)

현재 방식의 문제점:

👉 `"아아"` ≠ `"아메리카노"` (인식 불가)

📌 이유:

* TF-IDF는 **문자 기반 방식**
* 의미 이해 ❌
* 줄임말 / 동의어 처리 ❌

---

## 🔥 개선 방법 (필수 업그레이드)

### ✔ 방법 1: 동의어 사전 추가 (추천 ⭐)

```python
# 동의어 사전
synonym_dict = {
    "아아": "아이스 아메리카노",
    "뜨아": "따뜻한 아메리카노"
}

def preprocess_query(query):
    if query in synonym_dict:
        return synonym_dict[query]
    return query
```

👉 적용 예시:

```python
query = "아아"
query = preprocess_query(query)

results = recommend_products(query)
print(results)
```
